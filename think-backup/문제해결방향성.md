# 문제해결 방향성 (Needs Statement)

## 문제가설
생성형 AI를 업무에 써보고 싶은데, 제대로 배우지 못해서 시간만 낭비할까봐 두렵고, 동료들보다 뒤처져서 회사에서 경쟁력을 잃을까봐 불안하다

## 근본문제 분석
직장인들이 생성형 AI를 업무에 활용하고자 할 때 겪는 핵심 문제:
- **학습 효율성에 대한 불안**: 어떤 AI를 선택해야 할지 모르고, 배우는 데 시간이 오래 걸릴 것 같아 시작조차 두려움
- **상대적 역량 격차 우려**: 동료들은 이미 AI를 잘 활용하는 것 같은데, 자신만 뒤처져 있다는 불안감
- **업무 경쟁력 상실 공포**: AI를 못 쓰면 회사에서 도태될 것 같은 위기의식

이러한 불안과 두려움의 근본 원인은 **자신의 업무 목적에 맞는 적절한 생성형 AI를 찾지 못하고, 효과적으로 학습하고 적용하는 방법을 모르기 때문**입니다.

## Needs Statement

**업무 목적에 적합한 생성형 AI를 빠르게 찾고 싶은 직장인은 업무 경쟁력을 유지하고 생성형 AI를 효과적으로 활용하기 위하여, 자신의 상황에 맞는 AI를 확신을 갖고 선택하고 즉시 업무에 적용할 수 있는 명확한 가이드와 학습 경험이 필요하다.**

## 방향성 설명

### 핵심 가치
이 Needs Statement는 단순히 'AI 정보 제공'이 아니라, **불안감 해소와 자신감 회복**을 통한 경험 향상에 초점을 맞춥니다.

### 경험 향상의 요소
1. **확신을 갖고 선택**: 무엇을 배워야 할지 명확하게 알 수 있어 시간 낭비에 대한 두려움 해소
2. **즉시 업무 적용**: 배운 내용을 바로 실무에 쓸 수 있어 학습의 실효성 확보
3. **명확한 가이드**: 단계별로 무엇을 해야 하는지 알 수 있어 막막함 해소
4. **맞춤형 학습 경험**: 자신의 수준과 상황에 맞는 정보를 받아 효율적 학습 가능

### 기대 효과
- **심리적 안정**: 두려움과 불안감이 줄어들고 AI 학습에 대한 자신감 회복
- **시간 효율성**: 시행착오를 줄이고 필요한 AI를 빠르게 찾아 학습 시작
- **실질적 성과**: 배운 내용을 즉시 업무에 적용하여 가시적인 생산성 향상 경험
- **경쟁력 확보**: 동료 대비 뒤처지지 않고, 오히려 앞서나갈 수 있다는 자신감 획득
