# 서비스 기획서 발표 스크립트
**Title: GenAI-Info - 업무용 생성형 AI 정보제공 서비스**

---

## 1. 고객군 (Customer Segments)

**목표 고객군**
- 3년 이상 경력 직장인
- 생성형 AI 도입 필요성 인지하나 적절한 도구 선택 어려움
- 시행착오로 인한 시간/비용 낭비 우려

**얼리어답터**
- 마케터/개발자/기획자 직군 30-40대
- ChatGPT 등 기본 사용 경험 보유
- 팀 리더급, 의사결정 영향력 보유

---

## 2. 문제 (Problem)

**현재 문제**
- 수백 가지 AI 도구 중 업무에 맞는 선택 어려움
- 잘못된 선택으로 시간/비용 낭비 우려
- AI 역량 부족에 대한 경쟁력 저하 불안
- 도구별 프롬프트 학습 부담, 실전 예시 부족

**기존 대안**
- YouTube 튜토리얼: 시간 소모, 일반론적
- 유료 온라인 강의: 업무 맥락과 거리
- 동료 추천: 주관적, 객관성 부족
- 공식 문서: 전문 용어, 가파른 학습곡선
- 블로그/커뮤니티: 정보 산재, 검증 어려움

---

## 3. 고유가치제안 (Unique Value Proposition)

**고유가치제안**
- 직무별 실무 검증된 AI 도구 + 즉시 사용 가능한 프롬프트 템플릿
- 30분 Quick Start로 바로 실무 적용
- GitHub 스타일 버전 관리로 최신 정보 유지
- 한 장 비주얼 치트시트 제공
- 무료/저렴한 가격, 시행착오 없이 확신있는 선택

**High-level Concept**
"업무용 AI의 위키피디아 - 직무별로 검증된 AI 도구를 30분 만에 마스터"

---

## 4. 솔루션 (Solution)

- **직무별 AI 스타터 킷**: Top 3 도구 + 30개 프롬프트 템플릿, 한장 치트시트
- **GitHub 기반 커뮤니티**: Pull Request 방식 공동 개선, 별점/후기 검증
- **30분 Quick Start 가이드**: 도구 가입부터 첫 결과물 생성까지 단계별 안내
- **실무 사례 DB**: Before/After 사례, ROI 측정 가능

---

## 5. 경쟁우위 (Unfair Advantage)

- **오픈소스 자가 성장 생태계**: 전세계 실무자 자발적 기여, 경쟁사 추격 어려움
- **독특한 큐레이션 방식**: 한 장 치트시트 포맷, 극도 단순화 노하우
- **직무별 세분화 전략**: 범용 교육과 차별화, 업무 맥락 깊은 이해
- **Freemium 전략**: 진입 장벽 최소화 + 지속가능한 수익 모델

---

## 6. 수익구조 (Revenue Streams)

**프리미엄 서비스**
- 기본 리소스 라이브러리: 완전 무료
- 기업용 팀 라이선스: 월 9,900원/인 (5인 이상 할인)
- 직무별 심화 패키지: 건당 29,000원
- 기업 맞춤 컨설팅: 300만원~1,000만원
- 광고 및 제휴 수익

**목표 매출**
- 1차년도 1억원 / 2차년도 5억원 / 3차년도 15억원

---

## 7. 비용구조 (Cost Structure)

**고정비**
- 인건비: 핵심 팀 4명, 월 2,000만원
- 기술 인프라: 월 50만원 (클라우드 30만원 + 도구 20만원)
- 사무실: 월 50만원 (공유오피스) 또는 리모트

**변동비**
- 마케팅: 초기 월 50만원 → 최대 월 200만원 (CAC 5만원 이하)
- 콘텐츠 제작: 월 100만원

**초기 6개월 운영 비용: 총 1억 5천만원**

---

## 8. 핵심지표 (Key Metrics)

**AARRR 프레임워크**
- **Acquisition**: 월 신규 5천명 → 3개월 2만명, 오가닉 40%
- **Activation**: 리소스 사용 전환율 30%, 24시간 내 첫 사용 50%
- **Retention**: 1개월 재방문 40%, 3개월 재방문 25%
- **Revenue**: 유료 전환 5%, LTV 15만원, MRR 성장률 20%
- **Referral**: 월 기여자 10명 이상, K-factor 1.0 목표

---

## 9. 채널 (Channels)

- **오픈소스 커뮤니티**: GitHub 저장소, Awesome List, Reddit
- **직장인 플랫폼**: 블라인드, 디스콰이엇, 리멤버, 커리어리
- **콘텐츠 마케팅**: 브런치/미디엄/벨로그, YouTube 쇼츠/인스타 릴스
- **전문가 네트워크**: 인플루언서/강사 협업, 제휴 프로그램, 기업 교육 플랫폼
