# 생성형 AI 정보제공 서비스 - 유저스토리

## 개요
'업무 목적에 적합한 생성형 AI를 빠르게 찾고 싶은 직장인'을 위한 생성형AI 정보제공 서비스의 유저스토리입니다.
시퀀스 다이어그램 "AI 도구 발견 플로우"를 분석하여 마이크로서비스 기반으로 구성하였습니다.

## 마이크로서비스 구성
1. **User** - 사용자 인증 및 관리
2. **Analysis** - 업무 분석 및 니즈 분석 
3. **Recommendation** - AI 도구 추천 엔진
4. **Tool** - AI 도구 정보 관리 및 외부 API 연동
5. **Evaluation** - 만족도 평가 및 피드백 관리
6. **WebInterface** - 웹 사용자 인터페이스
