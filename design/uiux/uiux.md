## 프로토타입 화면 목록 정의

### Phase 1 (MVP) - M급 우선순위 화면
1. **01-로그인** - 사용자 인증 (UFR-USER-020)
2. **02-회원가입** - 신규 사용자 등록 (UFR-USER-010)
3. **03-대시보드** - 메인 화면 및 개인화 추천 (UFR-RECOM-020)
4. **04-도구목록** - 업무도메인별 도구 탐색 (UFR-TOOL-010)
5. **05-도구상세** - 도구 정보 상세 조회 (UFR-TOOL-030)
6. **06-도구필터** - 조건별 필터링 및 정렬 (UFR-TOOL-020)
7. **07-프로필설정** - 개인화 프로필 관리 (UFR-USER-030)

### Phase 2 (Core Features) - S급 기능 화면
8. **08-도구비교** - 다중 도구 비교 분석 (UFR-COMP-010, UFR-COMP-020)
9. **09-후기목록** - 사용후기 조회 (UFR-REVIEW-020)
10. **10-후기작성** - 후기 작성 및 평가 (UFR-REVIEW-010)
11. **11-즐겨찾기** - 관심도구 관리 (UFR-TOOL-040)
12. **12-사용가이드** - 도구별 사용 가이드 (UFR-TOOL-050)
13. **13-활용사례** - 업무별 활용 사례 (UFR-TOOL-060)
14. **14-알림센터** - 알림 확인 및 관리 (UFR-NOTI-020)
15. **15-마이페이지** - 개인 활동 이력 관리

### 공통 UI 컴포넌트
- **헤더** - 로고, 검색, 사용자 메뉴, 알림
- **네비게이션** - 메인 메뉴 및 브레드크럼
- **푸터** - 서비스 정보 및 링크
- **사이드바** - 카테고리 필터 및 부가 기능

## 화면 간 사용자 플로우 정의

### 신규 사용자 온보딩 플로우
```
01-로그인 → 02-회원가입 → 07-프로필설정 (가이드 투어 시작)
→ 03-대시보드 (개인화 추천 결과 확인) → 온보딩 가이드 툴팁
→ 04-도구목록 (필터 사용법 안내) → 05-도구상세 (기능 설명)
→ 11-즐겨찾기/바로가기 (첫 도구 사용 완료) → 온보딩 완료
```

**온보딩 가이드 구성:**
- 단계별 진행률 표시 (5단계)
- 각 화면별 핵심 기능 툴팁 (최대 3개)
- 건너뛰기 옵션 제공
- 완료 시 개인화 추천 품질 향상 안내

### 기존 사용자 일반 사용 플로우
```
01-로그인 → 03-대시보드 → 개인화 추천 확인 
→ 필요 시 04-도구목록/06-도구필터 → 05-도구상세 → 액션 수행
```

### 도구 비교 및 선택 플로우
```
04-도구목록 → 다중 선택 → 08-도구비교 → 상세 분석 확인 
→ 비교 결과 저장/공유 → 최종 도구 선택
```

### 커뮤니티 참여 플로우
```
05-도구상세 → 09-후기목록 → 후기 상호작용 
→ 10-후기작성 (필요 시) → 커뮤니티 기여
```

### 학습 및 활용 플로우
```
05-도구상세 → 12-사용가이드 → 13-활용사례 확인 
→ 바로가기로 실제 도구 사용
```

## 화면별 상세 설계

### 01-대시보드
**개요:** 
- 목적: 개인화된 AI 도구 추천 및 서비스 메인 허브 제공
- 관련 유저스토리: UFR-RECOM-020, UFR-TOOL-010
- 비즈니스 중요도: 매우 높음 (서비스 첫인상 결정)

**주요 기능:**
- 개인화 맞춤 추천 도구 표시 (상위 5-8개)
- 업무도메인별 인기 도구 바로가기
- 최근 조회한 도구 히스토리
- 즐겨찾기한 도구 빠른 접근
- 새로운 도구 및 업데이트 알림

**UI 구성요소:**
- Hero Section: 환영 메시지 및 주요 CTA 버튼
- 개인화 추천 카드 영역: 도구 썸네일, 이름, 추천 이유, 매칭도
- 카테고리별 인기 도구 그리드: 6-8개 주요 카테고리
- 최근 활동 사이드바: 조회 이력, 즐겨찾기, 알림
- 통계 위젯: 이번 주 신규 도구, 내 활동 요약

**인터랙션:**
- 추천 도구 카드 클릭: 도구 상세페이지로 이동
- 추천 새로고침: 다른 추천 결과 요청
- 카테고리 클릭: 해당 카테고리 도구 목록으로 이동
- 추천 피드백: 👍👎 버튼으로 즉석 평가

### 02-회원가입
**개요:**
- 목적: 신규 사용자의 원활한 서비스 가입 지원
- 관련 유저스토리: UFR-USER-010
- 비즈니스 중요도: 높음 (신규 사용자 확보)

**주요 기능:**
- 기본 회원 정보 입력 및 검증
- 업무도메인 선택 (개인화 기초 데이터)
- 이용약관 및 개인정보 처리방침 동의
- 실시간 입력 유효성 검증

**UI 구성요소:**
- 진행 단계 표시기: 3단계 (정보입력 → 도메인선택 → 완료)
- 입력 폼 그룹: 이름, 이메일, 비밀번호, 비밀번호 확인
- 업무도메인 선택 카드: 마케팅, 개발, 디자인, 기획, 영업, HR
- 약관 동의 체크박스: 필수/선택 동의 구분
- CTA 버튼: 가입하기 (유효성 통과 시 활성화)

**인터랙션:**
- 실시간 검증: 입력 완료 즉시 유효성 피드백
- 도메인 선택: 카드 클릭으로 선택 상태 변경
- 약관 보기: 모달 또는 새 탭으로 약관 내용 표시
- 소셜 가입: 구글/네이버/카카오 간편 가입 옵션

### 03-도구목록  
**개요:**
- 목적: 효율적인 AI 도구 탐색 및 발견 지원
- 관련 유저스토리: UFR-TOOL-010, UFR-TOOL-020
- 비즈니스 중요도: 매우 높음 (핵심 서비스 기능)

**주요 기능:**
- 업무도메인별 도구 목록 표시
- 다중 조건 필터링 (가격, 난이도, 평점, 카테고리)
- 정렬 옵션 (인기순, 평점순, 가격순, 최신순)
- 무한 스크롤 또는 페이지네이션
- 즐겨찾기 및 비교함 추가 기능

**UI 구성요소:**
- 상단 필터 바: 주요 필터 옵션 바로가기
- 사이드 필터 패널: 상세 필터링 옵션
- 도구 카드 그리드: 3-4열 반응형 레이아웃
- 정렬/보기 옵션: 그리드/리스트 뷰 토글
- 결과 요약: "총 XX개 도구가 조건에 맞습니다"

**인터랙션:**
- 필터 적용: 실시간 결과 업데이트
- 도구 카드 호버: 추가 정보 툴팁 표시
- 즐겨찾기 토글: 즉시 추가/제거 및 피드백
- 비교함 추가: 최대 5개까지 선택 상태 관리

### 04-도구상세
**개요:**
- 목적: 도구에 대한 종합적이고 상세한 정보 제공
- 관련 유저스토리: UFR-TOOL-030, UFR-TOOL-050, UFR-TOOL-060
- 비즈니스 중요도: 매우 높음 (구매 결정 지점)

**주요 기능:**
- 도구 기본정보 및 상세 스펙
- 가격 플랜별 기능 비교
- 사용자 평가 및 후기 요약
- 사용 가이드 및 활용 사례
- 바로가기 및 액션 버튼

**UI 구성요소:**
- 헤더 영역: 도구명, 로고, 평점, 주요 태그
- 탭 네비게이션: 개요/기능/가격/후기/가이드/사례
- 주요 정보 카드: 핵심 기능, 지원 플랫폼, 업데이트 정보
- 가격 비교표: 플랜별 기능 매트릭스
- 액션 버튼 그룹: 바로가기, 즐겨찾기, 비교함 추가, 공유

**인터랙션:**
- 탭 전환: 부드러운 애니메이션으로 콘텐츠 전환
- 가격 플랜 비교: 하이라이트 및 추천 표시
- 후기 필터링: 평점별, 업무별 후기 필터링
- 바로가기 클릭: 새 탭에서 외부 도구 열기 (사용 로그 기록)

### 05-도구비교
**개요:**
- 목적: 다중 도구 객관적 비교분석을 통한 의사결정 지원
- 관련 유저스토리: UFR-COMP-010, UFR-COMP-020, UFR-COMP-030
- 비즈니스 중요도: 높음 (차별화된 핵심 기능)

**주요 기능:**
- 최대 5개 도구 동시 비교
- 가격/기능/성능/사용성 다면적 비교
- 비교 결과 저장 및 공유
- 최적 도구 추천 알고리즘

**UI 구성요소:**
- 도구 선택 영역: 선택된 도구 카드 (제거 가능)
- 비교 매트릭스 테이블: 항목별 점수화 및 시각화
- 종합 점수 차트: 레이더 차트 또는 막대 그래프
- 추천 결과 배너: "당신에게 가장 적합한 도구"
- 액션 버튼: 저장, 공유, PDF 내보내기

**인터랙션:**
- 도구 추가/제거: 드래그 앤 드롭 또는 클릭으로 관리
- 비교 항목 가중치: 사용자가 중요도 조절 가능
- 상세 분석 토글: 항목별 상세 정보 확장/축소
- 결과 저장: 개인 계정에 비교 히스토리 저장

### 06-후기작성
**개요:**
- 목적: 사용자 경험 공유를 통한 커뮤니티 기여 유도
- 관련 유저스토리: UFR-REVIEW-010
- 비즈니스 중요도: 중간 (커뮤니티 활성화)

**주요 기능:**
- 다면적 평가 시스템 (기능성, 사용성, 가격 만족도)
- 구조화된 후기 작성 (좋은 점, 아쉬운 점, 총평)
- 사용 정보 입력 (목적, 기간, 빈도)
- 추천 여부 및 대상 설정

**UI 구성요소:**
- 평가 영역: 별점 및 세부 평가 슬라이더
- 사용 정보 선택: 드롭다운 및 라디오 버튼
- 텍스트 입력: 구조화된 텍스트 영역 (글자수 제한 표시)
- 미리보기 모드: 작성 중인 후기 실시간 미리보기
- 제출 버튼: 필수 항목 완료 시 활성화

**인터랙션:**
- 별점 호버 이펙트: 마우스 오버 시 평점 미리보기
- 글자수 실시간 카운트: 제한 글자수 대비 진행률 표시
- 임시 저장: 작성 중인 내용 자동 임시 저장
- 작성 취소: 확인 다이얼로그 후 이전 페이지로 복귀

### 07-프로필설정
**개요:**
- 목적: 개인화 추천 품질 향상을 위한 사용자 정보 관리
- 관련 유저스토리: UFR-USER-030
- 비즈니스 중요도: 높음 (개인화 품질 결정)

**주요 기능:**
- 업무 정보 설정 (도메인, 경력, 관심분야)
- 추천 선호도 설정 (가격대, 도구 유형)
- 알림 및 개인정보 설정
- 계정 보안 관리

**UI 구성요소:**
- 탭 메뉴: 기본정보/선호도/알림설정/계정보안
- 프로필 이미지: 아바타 업로드 또는 기본 이미지 선택
- 설정 카드 그룹: 카테고리별 설정 항목 그룹화
- 토글 스위치: 알림 설정 등 ON/OFF 토글
- 저장 버튼: 변경사항 저장 및 피드백

**인터랙션:**
- 실시간 저장: 설정 변경 시 즉시 저장 또는 명시적 저장
- 설정 가이드: 각 설정의 영향도 툴팁으로 안내
- 미리보기: 선호도 설정 기반 추천 결과 미리보기
- 계정 연동: 소셜 계정 연동/해제 관리

## 화면간 전환 및 네비게이션

### 메인 네비게이션 구조
```
헤더 (Global Navigation)
├── 로고 (홈으로)
├── 검색바 (전역 도구 검색)
├── 주메뉴
│   ├── 도구 탐색 (04-도구목록)
│   ├── 추천 (03-대시보드)
│   ├── 비교분석 (08-도구비교)
│   └── 커뮤니티 (09-후기목록)
└── 사용자 메뉴
    ├── 알림 (14-알림센터)
    ├── 즐겨찾기 (11-즐겨찾기)
    ├── 프로필 (07-프로필설정)
    └── 로그아웃
```

### 브레드크럼 네비게이션
- 깊이 2단계 이상 페이지에서 제공
- 클릭 가능한 경로 표시로 빠른 역방향 이동 지원
- 예: `홈 > 도구 탐색 > 마케팅 도구 > ChatGPT`

### 페이지 전환 애니메이션
- **페이드 인/아웃**: 일반적인 페이지 전환
- **슬라이드**: 순차적 진행 (회원가입 단계)
- **모달 오버레이**: 빠른 액션 (후기 작성, 설정 변경)
- **탭 전환**: 콘텐츠 영역만 부드럽게 전환

### 상황별 네비게이션 패턴
- **첫 방문자**: 온보딩 가이드를 통한 단계별 안내
- **로그인 사용자**: 개인화된 추천 중심 네비게이션
- **도구 탐색 중**: 필터 상태 유지 및 빠른 비교 접근
- **비교 진행 중**: 비교 상태 유지 및 관련 액션 우선 노출

## 반응형 설계 전략

### 브레이크포인트 정의
- **Mobile**: 320px - 767px (세로형 모바일 우선)
- **Tablet**: 768px - 1023px (태블릿 및 작은 노트북)
- **Desktop**: 1024px - 1439px (일반 데스크톱)
- **Large Desktop**: 1440px 이상 (대형 모니터)

### 레이아웃 전략
**Mobile-First 접근법**
- 핵심 기능 우선, 점진적 기능 확장
- 터치 인터페이스 최적화 (버튼 크기 44px 이상)
- 단일 컬럼 레이아웃 중심

**콘텐츠 우선순위**
- 모바일: 검색 → 개인화 추천 → 주요 도구
- 태블릿: 검색 + 필터 → 2열 그리드 → 사이드바
- 데스크톱: 전체 UI → 3-4열 그리드 → 완전한 사이드바

### 반응형 컴포넌트 설계
**네비게이션**
- 모바일: 햄버거 메뉴 + 하단 탭바
- 태블릿: 축약된 상단 메뉴
- 데스크톱: 완전한 상단 네비게이션

**도구 목록**
- 모바일: 1열 카드 리스트
- 태블릿: 2열 그리드
- 데스크톱: 3-4열 그리드

**도구 상세**
- 모바일: 세로 스택 레이아웃
- 태블릿: 2열 (정보 + 액션)
- 데스크톱: 3영역 (정보 + 상세 + 액션)

## 접근성 보장 방안

### 웹 접근성 표준 준수 (WCAG 2.1 AA)
**인식의 용이성 (Perceivable)**
- 색상에만 의존하지 않는 정보 전달
- 명도 대비 4.5:1 이상 확보
- 이미지 대체텍스트 제공
- 동영상 자막 및 음성해설 지원

**운용의 용이성 (Operable)**
- 키보드만으로 모든 기능 사용 가능
- 포커스 표시 명확화
- 자동 재생 콘텐츠 제어 기능
- 발작 유발 요소 제거

**이해의 용이성 (Understandable)**
- 일관된 네비게이션 구조
- 명확한 라벨 및 지시사항
- 오류 메시지의 구체적 안내
- 예측 가능한 인터페이스 동작

**견고성 (Robust)**
- 시맨틱 HTML 마크업
- ARIA 레이블 적절한 사용
- 스크린 리더 최적화
- 다양한 보조기술 호환성

### 접근성 강화 기능
**시각적 접근성**
- 고대비 모드 제공
- 글꼴 크기 조절 (100% - 200%)
- 색각 이상자 고려 색상 팔레트
- 화면 확대 도구 호환성

**청각적 접근성**
- 모든 음성 콘텐츠 텍스트 대안 제공
- 알림음 시각적 표시 병행
- 소리 설정 세밀 제어

**인지적 접근성**
- 복잡한 작업 단계별 분할
- 명확하고 간단한 언어 사용
- 중요 정보 반복 제시
- 충분한 작업 완료 시간 제공

**물리적 접근성**
- 키보드 네비게이션 완벽 지원
- 터치 타겟 최소 크기 보장
- 드래그 앤 드롭 대안 제공
- 음성 인식 소프트웨어 호환

## 성능 최적화 방안

### 프론트엔드 최적화
**리소스 최적화**
- 이미지 최적화: WebP 포맷, 반응형 이미지, 지연 로딩
- CSS/JS 번들링 및 압축: 불필요한 코드 제거
- 폰트 최적화: 웹폰트 preload, font-display: swap
- CDN 활용: 정적 리소스 글로벌 배포

**렌더링 최적화**
- 가상 스크롤링: 대량 목록 성능 개선
- 컴포넌트 지연 로딩: 필요 시점에 로드
- 메모이제이션: 불필요한 리렌더링 방지
- Critical CSS: 초기 렌더링 속도 개선

**데이터 최적화**
- API 응답 캐싱: 브라우저 및 서비스 워커 캐시
- 페이지네이션/무한 스크롤: 데이터 분할 로딩
- 검색 디바운싱: 불필요한 API 호출 방지
- 이미지 플레이스홀더: 로딩 UX 개선

### 백엔드 최적화  
**데이터베이스 최적화**
- 쿼리 최적화 및 인덱스 설계
- 데이터베이스 연결 풀링
- 읽기 전용 복제본 활용
- 쿼리 결과 캐싱 (Redis)

**API 최적화**
- GraphQL을 통한 필요 데이터만 요청
- API 응답 압축 (gzip/brotli)
- HTTP/2 및 HTTP/3 지원
- API 버전 관리 및 캐싱 전략

### 성능 모니터링
**Core Web Vitals 최적화**
- LCP (Largest Contentful Paint) < 2.5s
- FID (First Input Delay) < 100ms  
- CLS (Cumulative Layout Shift) < 0.1

**성능 측정 도구**
- Google Lighthouse: 종합 성능 분석
- Google PageSpeed Insights: 실제 사용자 데이터
- Web Vitals Chrome Extension: 실시간 모니터링
- 사용자 정의 성능 메트릭: 비즈니스 중요 지표

**성능 목표 수립**
- 페이지 로딩 시간: 3초 이내
- API 응답 시간: 1초 이내
- 검색 결과 표시: 2초 이내
- 이미지 로딩 완료: 5초 이내
